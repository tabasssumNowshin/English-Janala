Frequently Asked Questions

1.What is the difference between var, let and const?
var: Function-scoped, re-declarable, accessible before declaration but value undefined.
let: Block-scoped, not accessible before declaration.
const: Block-scoped,not accessible before declaration.
2.Difference between map, filter and foreach method.
map: Transforms each array element and returns a new array.
filter: Returns a new array with elements that pass a condition.
forEach: Iterates over elements, performing a function, but returns undefined.
3.Arrow function and difference with normal function.
Arrow Function: A compact way to write functions using =>, and it doesn’t have its own this.
Normal Function: Uses the function keyword, has its own this, and can access the arguments object.
4.How promise work?
A Promise works by handling asynchronous tasks, it has three states: pending, resolved, or rejected, allowing you to use .then() and .catch().
5.How closures work in JavaScript?
A closure works by allowing a function to remember and access variables from its parent scope, even after the parent function has finished executing.
